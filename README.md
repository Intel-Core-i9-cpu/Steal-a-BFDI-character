# Steal-a-BFDI-character

A starter browser game inspired by **Steal a Brainrot**, but using BFDI characters.

## Current prototype includes
- Rolling for random BFDI characters with weighted rarities.
- Passive income generated by your current team.
- A rival team that you can attempt to steal from.
- Steal cooldown + failure penalty to create risk/reward gameplay.

## Run locally
Because this is plain HTML/CSS/JS, you can run it with any static file server.

Examples:

```bash
python3 -m http.server 8000
# then open http://localhost:8000
```

or open `index.html` directly in your browser.
